# API Contract (Initial)

## Authentication

- `POST /auth/register/`
- `POST /auth/login/`
- `POST /auth/refresh/`
- `GET /auth/me/`

## Vehicles

- `GET /vehicles/`

## Quotas

- `GET /quotas/`
- `GET /quotas/{id}/`

## Transactions

- `GET /transactions/`
- `POST /transactions/`

## Stations

- `GET /stations/`
- `POST /stations/`
