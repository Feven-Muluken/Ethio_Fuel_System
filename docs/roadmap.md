# Roadmap (Prototype to Production)

## MVP (Phase 1)

- Buyer registration/login.
- Vehicle registration.
- Quota lookup.
- QR/plate validation endpoint.
- Transaction creation and quota deduction.

## Phase 2

- Station operations dashboard with improved UX.
- Regulator dashboard with live summaries.
- Role-based authorization hardening.
- Exportable reports (CSV/PDF).

## Phase 3

- Fraud detection rules (frequency, station anomalies, duplicate scans).
- Alerting pipelines for regulators.
- Audit trail and immutable event history.

## Phase 4

- USSD channel integration.
- Mobile money integration.
- Multi-language support and rural connectivity resilience.
